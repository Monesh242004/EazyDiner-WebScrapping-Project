# 🍽️ EazyDiner Restaurant Data Analysis & Web Scraping

A complete Data Analytics project that collects restaurant data from the EazyDiner website using Python Web Scraping techniques and performs Exploratory Data Analysis (EDA) to uncover valuable insights about restaurants, ratings, cuisines, pricing, and customer preferences.

---

## 📌 Project Overview

This project demonstrates an end-to-end data analytics workflow:

- Scraped restaurant information from the EazyDiner website.
- Cleaned and prepared the collected dataset.
- Performed Exploratory Data Analysis (EDA).
- Created meaningful visualizations.
- Generated business insights for restaurant trends and customer behavior.

---

## 🎯 Problem Statement

Restaurant platforms contain a large amount of valuable information, but the data is often unstructured and difficult to analyze.

The objective of this project is to scrape restaurant data from EazyDiner, clean the dataset, and analyze various factors such as ratings, cuisines, cost, and locations to identify useful business insights.
